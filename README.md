# Township Tycoon

A vertical scrolling idle-management game where you build a factory empire.

## Features

- **5 Unique Zones**: Corn, Tomato, Dairy, Orchard, Chocolate.
- **Manager System**: Hire staff for buffs and automation.
- **Dynamic Market**: Prices change based on random events (Surges/Crashes).
- **Day/Night Cycle & Weather**: Unlocked via themes.
- **Prestige System**: Earn Golden Keys to unlock powerful upgrades.

## Tech Stack

- React
- TypeScript
- Tailwind CSS

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

## Note

This game uses a single-file component architecture for easy integration and portability.
